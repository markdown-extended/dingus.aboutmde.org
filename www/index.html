  **MDE** _dingus_
a markdown-extended web editor

Use the field above to write your Markdown-Extended content then click on the eye icon (it appears when the mouse is hover the field) to get the result. You can also use the fullscreen feature by clicking on the second icon. To get help about the MDE syntax, you can read the Markdown syntax reminders.

This editor is a custom version of the original EpicEditor to use the PHP MarkdownExtended parser. It uses the MDE-Service PHP interface to post the markdown content of the editor (via a synchronous XMLHttpRequest) at each run (the original content is not parsed in JavaScript). You can find the sources of the editor online at http://github.com/piwi/mde-editor.

This page is built with the help of jQuery, Bootstrap and Font Awesome.
